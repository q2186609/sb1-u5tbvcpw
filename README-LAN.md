# 贪吃蛇游戏局域网部署指南

## 🌐 局域网访问配置

本项目已配置为支持局域网访问，允许同一网络下的其他设备连接并游戏。

### 📱 访问地址

当前配置的局域网IP：`192.168.2.80`

- **前端游戏界面**: http://192.168.2.80:5173
- **WebSocket服务器**: ws://192.168.2.80:3001

### 🚀 快速启动

#### 方法一：使用启动脚本（推荐）
```bash
./start-lan.sh
```

#### 方法二：手动启动
```bash
# 启动后端服务器
cd server
npm start

# 在新终端启动前端（在项目根目录）
npm run dev
```

#### 方法三：同时启动
```bash
npm run start:all
```

### 🔧 网络配置说明

#### 服务器配置
- **监听地址**: `0.0.0.0` (所有网卡)
- **后端端口**: `3001`
- **前端端口**: `5173`

#### CORS配置
服务器已配置支持以下来源：
- http://localhost:5173
- http://127.0.0.1:5173  
- http://192.168.2.80:5173
- 所有192.168.x.x网段
- 所有10.x.x.x网段（企业网络）
- 所有172.16-31.x.x网段（私有网络）

#### Socket连接配置
客户端会自动检测当前域名并连接到对应的Socket服务器：
- 本地访问时连接到 `localhost:3001`
- 局域网访问时连接到 `当前IP:3001`

### 📋 使用步骤

1. **启动服务**
   ```bash
   ./start-lan.sh
   ```

2. **本地访问**
   - 打开浏览器访问：http://localhost:5173
   - 或访问：http://127.0.0.1:5173

3. **局域网访问**
   - 其他设备访问：http://192.168.2.80:5173
   - 确保设备在同一局域网内

4. **多人游戏**
   - 创建房间或加入现有房间
   - 支持快速匹配功能
   - 支持房间密码保护

### 🛠️ 自定义IP地址

如果你的局域网IP不是 `192.168.2.80`，可以通过以下方式配置：

#### 方法一：修改环境变量
编辑 `.env` 文件：
```env
VITE_SOCKET_URL=http://你的IP地址:3001
```

#### 方法二：修改启动脚本
编辑 `start-lan.sh` 脚本，更新LOCAL_IP变量

#### 方法三：自动检测
启动脚本会自动检测本地IP地址，通常无需手动配置

### 🔍 获取本地IP地址

#### macOS/Linux
```bash
ifconfig | grep "inet " | grep -v 127.0.0.1 | awk '{print $2}' | head -1
```

#### Windows
```cmd
ipconfig | findstr "IPv4"
```

### 🐛 故障排除

#### 无法连接到服务器
1. 检查防火墙设置，确保端口3001和5173开放
2. 确认设备在同一局域网内
3. 检查IP地址配置是否正确

#### Socket连接失败
1. 检查 `.env` 文件中的VITE_SOCKET_URL配置
2. 确认后端服务器正常运行
3. 查看浏览器控制台错误信息

#### 无法访问游戏界面
1. 确认前端服务正常启动
2. 检查端口5173是否被占用
3. 尝试使用不同的浏览器

### 💡 提示

1. **建议使用Chrome或Safari浏览器**以获得最佳游戏体验
2. **确保网络稳定**以避免游戏中断
3. **关闭不必要的网络应用**以减少延迟
4. **移动设备**也可以通过浏览器访问游戏
